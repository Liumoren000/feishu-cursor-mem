# 中药产业全生命周期追溯系统（PRD）

## 1. 项目背景与目标
中药材质量直接关系到临床疗效和用药安全。本项目旨在构建一个覆盖“种植-加工-质检-仓储-流通”全生命周期的追溯系统，通过“一物一码”技术，实现来源可查、去向可追、责任可究，满足监管要求并提升品牌公信力。

## 2. 核心业务流程
1. **种植基地管理**：记录种子来源、施肥记录、喷药记录、采收时间。
2. **生产加工管理**：清洗、切片、炮制、烘干、包装记录（符合 GMP 标准）。
3. **品质检测中心**：上传第三方质检报告（农残、重金属、成分含量）。
4. **赋码关联**：生成追溯码并与批次/单品数据绑定。
5. **流通打点**：仓库入库、分拨出库、门店接收。
6. **扫码展示**：消费者或监管部门扫码查看全链路报告。

## 3. 功能需求详情

### 3.1 种植端（移动端/后台）
- **地块管理**：基地经纬度、土壤质量背景。
- **农事记录**：拍照上传施肥、灌溉、除草、除虫操作，记录人、时间、投入品。
- **采收管理**：记录采收批次、初选重量、采收环境。

### 3.2 加工端（Web/移动端）
- **原料验收**：关联种植端批次。
- **炮制加工**：记录工艺参数（时间、温度、添加辅料）、设备编号。
- **赋码打印**：支持热敏打印、激光赋码，支持大包码（箱码）与小包码（个码）嵌套。

### 3.3 质检模块
- **报告上传**：支持 PDF/图片上传。
- **数据结构化**：录入关键指标（如：黄曲霉毒素、二氧化硫残留量等）。
- **合格判定**：质检合格后方可生成流通码。

### 3.4 追溯码中心
- **码段管理**：规则自定义（年份+产品ID+流水号+防伪位）。
- **防伪策略**：首次扫码提示、重复扫码预警（显示扫码次数及地点）。

### 3.5 后台管理（Web端）
- **供应链图谱**：可视化展示某批次药材的流转路径。
- **库存预警**：效期提醒、呆滞料提醒。
- **统计分析**：各基地产量对比、质检合格率统计。

## 4. 消费者扫码页（H5/小程序）
- **动态交互**：以时间轴形式展示药材的“前世今生”。
- **产地视频**：支持基地摄像头实时画面或录像回放。
- **一键验证**：显示真伪校验结果。

## 5. 非功能性需求
- **数据安全**：关键数据上链（可选，提升不可篡改性）。
- **并发性能**：支持高峰期（如扫码抽奖活动）高并发访问。
- **易用性**：基地端支持断网环境下缓存农事记录。
